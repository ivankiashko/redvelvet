# Подробное описание функционала

## 1. Система авторизации

### Регистрация
- Выбор типа аккаунта (Клиент/Модель)
- Email и пароль
- Автоматическое сохранение в LocalStorage

### Вход
- Проверка по email
- Восстановление сессии при обновлении страницы

### Выход
- Подтверждение действия
- Очистка текущей сессии
- Сохранение данных платформы

## 2. Навигация

### Динамические кнопки

**Для гостя:**
- "Регистрация" → открывает модальное окно регистрации
- "Создать анкету" → предлагает регистрацию как модель

**Для клиента:**
- "Личный кабинет" → переход в ЛК клиента
- "К анкетам" → возврат к просмотру анкет

**Для модели (без анкеты):**
- "Создать анкету" → форма создания анкеты
- "К анкетам" → просмотр всех анкет

**Для модели (с анкетой):**
- "Моя анкета" → редактирование и статистика
- "К анкетам" → просмотр всех анкет

## 3. Создание анкеты модели

### Обязательные поля

**Основная информация:**
- Имя (псевдоним) - автоматическая стандартизация
- Возраст (18-99 лет)
- Город (выбор из списка)

**Параметры:**
- Рост (140-220 см)
- Вес (40-150 кг)
- Размер груди (1-6+)

**Внешность:**
- Цвет глаз (Карие, Зеленые, Голубые, Серые)
- Цвет волос (Блондинка, Брюнетка, Шатенка, Рыжая)
- Национальность (Славянка, Азиатка, Латиноамериканка, Африканка, Смешанная)

**Фигура:**
- Тип фигуры (Стройная, Спортивная, Аппетитная, Модельная)
- Размер одежды (XS-XL)

**Контент:**
- Описание анкеты (свободный текст)
- Услуги (список, каждая с новой строки)
- Цена (₽/час, минимум 1000)

**Медиа (опционально):**
- Фото (JPG, PNG)
- Видео (MP4)
- Максимальный размер файла: 10 МБ

### Автоматическая обработка

- Стандартизация имени: первая буква заглавная
- Стандартизация услуг: каждая услуга с заглавной буквы
- Валидация всех полей
- Превью загруженных медиа
- Возможность удаления медиа файлов

## 4. Система поиска и фильтрации

### Поиск по ключевым словам
- Поиск по имени модели
- Поиск по описанию
- Поиск по услугам

### Вкладки фильтров

#### Основные
- Возраст (от/до)
- Город
- Рейтинг (5★, 4+★, 3+★)

#### Внешность
- Цвет глаз
- Цвет волос
- Национальность
- Размер груди

#### Параметры
- Рост (от/до)
- Вес (от/до)

#### Услуги
- Цена (от/до)
- Тип услуг (автоматически из доступных)

### Особенности
- Все фильтры применяются автоматически
- Комбинирование нескольких фильтров
- Мгновенное обновление результатов

## 5. Просмотр анкет

### Карточка анкеты (превью)
- Главное фото
- Значок верификации (если есть)
- Имя и возраст
- Основные параметры (рост, грудь, цена)
- Рейтинг и количество отзывов
- Краткое описание (100 символов)
- Первые 3 услуги

### Детальный просмотр
- Все фотографии (галерея)
- Полная информация о параметрах
- Полное описание
- Все услуги
- Рейтинг и отзывы
- Кнопки: "Забронировать", "Оставить отзыв"
- Счетчик просмотров (увеличивается автоматически)

## 6. Система отзывов

### Ограничения
- Только для зарегистрированных пользователей
- Один отзыв от одного email на одну анкету

### Процесс создания
1. Выбор оценки (1-5 звезд)
2. Написание текста отзыва
3. Отправка

### Обработка
- Автоматический пересчет среднего рейтинга
- Обновление количества отзывов
- Сортировка по дате (новые первые)

### Отображение
- Оценка звездами
- Текст отзыва
- Дата публикации

## 7. Система оплаты

### Варианты оплаты

#### Криптовалюта
- Требуется привязка кошелька
- Поддерживаемые валюты: BTC, ETH, USDT, TON
- Инструкция по оплате
- Прямая отправка на адрес модели

#### По договоренности
- Прямая связь с моделью
- Обсуждение условий
- Любой удобный способ

## 8. Личный кабинет клиента

### Разделы

**Кошелек:**
- Привязка криптокошелька
- Отображение типа и адреса
- Возможность изменения

**Избранное:**
- Сохраненные анкеты (в разработке)

**Мои отзывы:**
- История всех оставленных отзывов (в разработке)

**Навигация:**
- Кнопка "К анкетам" - возврат к просмотру
- Кнопка "Выход" - выход из аккаунта

## 9. Личный кабинет модели

### Статистика
- Количество просмотров
- Средний рейтинг
- Количество отзывов
- Позиция в рейтинге

### Управление анкетой
- Редактирование всех полей
- Добавление/удаление медиа
- Сохранение изменений

### Кошелек
- Привязка криптокошелька для получения платежей
- Отображение адреса

### Навигация
- Кнопка "К анкетам" - просмотр других анкет
- Кнопка "Выход" - выход из аккаунта

## 12. Хранение данных

### LocalStorage
Текущее решение использует LocalStorage для хранения:
- Данных пользователя
- Анкет моделей
- Отзывов
- Настроек кошельков

### Структура данных

**Пользователь:**
```javascript
{
  email: "user@example.com",
  type: "client" | "model",
  wallet: {
    type: "BTC" | "ETH" | "USDT" | "TON",
    address: "wallet_address"
  }
}
```

**Профиль модели:**
```javascript
{
  id: 123456789,
  name: "Имя",
  age: 25,
  city: "moscow",
  height: 170,
  weight: 55,
  bustSize: "3",
  eyeColor: "Зеленые",
  hairColor: "Блондинка",
  nationality: "Славянка",
  bodyType: "Модельная",
  clothingSize: "S",
  description: "Описание...",
  services: ["Услуга 1", "Услуга 2"],
  price: 8000,
  images: [...],
  videos: [...],
  rating: 4.8,
  reviewCount: 12,
  views: 245,
  verified: true,
  createdAt: "2024-10-15T..."
}
```

**Отзыв:**
```javascript
{
  rating: 5,
  text: "Отличная модель!",
  date: "2024-10-30T...",
  userId: "user@example.com"
}
```

## 13. Безопасность

### Текущая реализация
⚠️ Упрощенная система без реальной безопасности

### Планируемые улучшения
- Хеширование паролей (bcrypt)
- JWT токены
- HTTPS обязательно
- Защита от XSS
- Защита от CSRF
- Rate limiting
- Капча при регистрации

## 14. Производительность

### Оптимизации
- Минимальное использование внешних библиотек
- Vanilla JavaScript для максимальной скорости
- CSS анимации вместо JavaScript
- Lazy loading для изображений (в планах)
- Кеширование данных в LocalStorage

### Рекомендации
- Регулярная очистка LocalStorage
- Оптимизация изображений перед загрузкой
- Ограничение размера медиа файлов (10 МБ)
