# RedVelvet Platform - Заметки для разработки

## ⚠️ ВАЖНО: Структура проекта

### Главный файл проекта:

**index.html** - главный файл платформы с модульной структурой

**При внесении изменений в HTML-структуру, формы, модальные окна или любые элементы интерфейса, необходимо также обновлять связанные CSS и JS файлы!**

### Чек-лист перед коммитом:

- [ ] Изменения протестированы в `index.html`
- [ ] Обновлены соответствующие стили в `src/css/styles.css`
- [ ] Обновлена логика в `src/js/app.js`
- [ ] Проверена работа на мобильных устройствах

## Структура проекта

```
redvelvet/
├── index.html                    # Главная страница проекта
├── src/
│   ├── css/
│   │   └── styles.css           # Общие стили
│   └── js/
│       └── app.js               # Основная логика приложения
└── admin/
    ├── index.html               # Админ-панель
    ├── admin.js                 # Логика админки
    └── admin-styles.css         # Стили админки
```
